1. The if Statement

The if statement checks a condition. If the condition is True, the code inside runs.

# Syntax
if condition:
    # code to execute if condition is True

Example 1: Check if a number is positive

num = 10
if num > 0:
    print("The number is positive")
Output: The number is positive

Example 2: Check age for voting

age = 18
if age >= 18:
    print("You are eligible to vote")
Output: You are eligible to vote

2. The if-else Statement

if-else allows you to run one block of code if the condition is True, and another block if it is False.

# Syntax
if condition:
    # code if True
else:
    # code if False

Example 1: Even or Odd

num = 7
if num % 2 == 0:
    print("Even")
else:
    print("Odd")
Output: Odd

Example 2: Pass or Fail

score = 45
if score >= 50:
    print("Pass")
else:
    print("Fail")
Output: Fail

Example 3: Real Life - Discount Eligibility

purchase = 500
if purchase >= 1000:
    discount = purchase * 0.1
    print(f"Discount: {discount}")
else:
    print("No discount. Spend 1000+ to get 10% off")
Output: No discount. Spend 1000+ to get 10% off

3. The elif Statement

elif (else if) allows you to check multiple conditions in sequence.

# Syntax
if condition1:
    # code
elif condition2:
    # code
elif condition3:
    # code
else:
    # code if none are True

Example 1: Grade Calculator

score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"
print(f"Grade: {grade}")
Output: Grade: B

Example 2: Traffic Light

light = "yellow"
if light == "red":
    print("Stop")
elif light == "yellow":
    print("Slow down")
elif light == "green":
    print("Go")
else:
    print("Invalid color")
Output: Slow down

Example 3: Ethiopian Coffee Strength

strength = 3
if strength == 1:
    print("Light roast - mild flavor")
elif strength == 2:
    print("Medium roast - balanced")
elif strength == 3:
    print("Dark roast - strong and bold")
else:
    print("Unknown strength level")
Output: Dark roast - strong and bold

4. Nested if Statements

You can put an if statement inside another if statement. This is called nesting.

# Syntax
if condition1:
    if condition2:
        # code when both are True

Example: Checking Login Credentials

username = "student"
password = "python123"

if username == "student":
    print("Username correct")
    if password == "python123":
        print("Login successful")
    else:
        print("Wrong password")
else:
    print("Username not found")
Output: Username correct Login successful

Example: Ethiopian Bus Ticket System

age = 25
has_id = True

if age >= 18:
    print("Adult passenger")
    if has_id:
        print("Ticket issued")
    else:
        print("Please show your ID")
else:
    print("Child ticket (50% discount)")
Output: Adult passenger Ticket issued

5. The for Loop

The for loop is used to iterate over a sequence (like a list, string, or range).

# Syntax
for variable in sequence:
    # code to repeat

Example 1: Print numbers 1-5

for i in range(1, 6):
    print(i)
1 2 3 4 5

Example 2: Loop through a list

names = ["Tsegaye", "Mulu", "Abdo"]
for name in names:
    print(f"Hello {name}")
Hello Tsegaye
Hello Mulu
Hello Abdo

Example 3: Real Life - Daily Savings

total = 0
for day in range(1, 8):  # 7 days
    total = total + 20
    print(f"Day {day}: {total} Birr")
Day 1: 20 Birr
Day 2: 40 Birr
Day 3: 60 Birr
Day 4: 80 Birr
Day 5: 100 Birr
Day 6: 120 Birr
Day 7: 140 Birr

6. The while Loop

The while loop repeats as long as a condition is True.

# Syntax
while condition:
    # code to repeat

Example 1: Count down

count = 5
while count > 0:
    print(count)
    count = count - 1
5 4 3 2 1

Example 2: Sum until 10

total = 0
num = 1
while total < 10:
    total = total + num
    num = num + 1
print(f"Total: {total}")
Total: 10

Example 3: Real Life - ATM Pin Attempts

correct_pin = "1234"
attempts = 3

while attempts > 0:
    pin = input("Enter PIN: ")
    if pin == correct_pin:
        print("Access granted")
        break
    else:
        attempts = attempts - 1
        print(f"Wrong. {attempts} attempts left")

7. break and continue

Control the flow of your loops with break and continue.

break - Stop the loop

for i in range(1, 10):
    if i == 5:
        break
    print(i)
1 2 3 4

continue - Skip to next iteration

for i in range(1, 6):
    if i == 3:
        continue
    print(i)
1 2 4 5

Real Life Example: Skip weekends, stop after Friday

days = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
for day in days:
    if day == "Sat":
        continue  # Skip Saturday
    if day == "Sun":
        continue  # Skip Sunday
    if day == "Fri":
        print("Friday - Last work day!")
        break
    print(f"{day} - Working")
Mon - Working
Tue - Working
Wed - Working
Thu - Working
Friday - Last work day!

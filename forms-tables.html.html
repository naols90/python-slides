What is a Form?

A form is used to collect information from users - like login, signup, search, or feedback.

What is a Table?

A table is used to display data in rows and columns - like timetables, marksheets, or price lists.

1. HTML Form Basics

The <form> tag is the container for all form elements.

<form action="submit.php" method="post">
    <!-- form elements go here -->
</form>

Important Form Attributes

Attribute What it does Example action Where to send the form data action="process.php" method How to send data (get or post) method="post"

Simple Form Example

<form>
    Name: <input type="text">
</form>
📺 This is what the user sees (pure HTML):
Name:

2. Text Inputs

The <input> tag is the most common form element.

Text Field

<input type="text" name="username">
Result:

✅ Used for: Name, username, search

Password Field

<input type="password" name="pwd">
Result:

✅ Hides characters with dots

Email Field

<input type="email" name="email">
Result:

✅ Checks for @ and .com

Complete Example with Labels

<form>
    <label>Full Name:</label>
    <input type="text" name="fullname">
    <br>
    <label>Email:</label>
    <input type="email" name="email">
    <br>
    <label>Password:</label>
    <input type="password" name="password">
</form>

4. Form Buttons

Submit Button

<input type="submit" value="Send">

✅ Sends form data to server

Reset Button

<input type="reset" value="Clear">

✅ Clears all form fields

Button (Custom)

<button type="button">Click Me</button>

✅ Used with JavaScript

Complete Registration Form Example

<form action="register.php" method="post">
    <label>Full Name:</label>
    <input type="text" name="name"><br>

    <label>Email:</label>
    <input type="email" name="email"><br>

    <label>Password:</label>
    <input type="password" name="pass"><br>

    <label>City:</label>
    <select name="city">
        <option>Addis Ababa</option>
        <option>Gondar</option>
    </select><br>

    <label>Message:</label>
    <textarea rows="3"></textarea><br>

    <input type="submit" value="Register">
    <input type="reset" value="Clear">
</form>

5. HTML Tables

Tables display data in rows and columns.

Table Tags

Tag Meaning <table> Creates a table <tr> Table Row <th> Table Header (bold, centered) <td> Table Data (regular cell)

Simple Table Example

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Almaz</td>
        <td>15</td>
    </tr>
    <tr>
        <td>Biruk</td>
        <td>16</td>
    </tr>
</table>
📺 This is what the user sees:
Name Age
Almaz 15
Biruk 16

📊 Student Marksheet

        <td>78</td>
        <td>88</td>
    </tr>

📅 Weekly Schedule

<table border="1">
    <tr>
        <th>Day</th>
        <th>Subject</th>
    </tr>
    <tr>
        <td>Monday</td>
        <td>HTML</td>
    </tr>
    <tr>
        <td>Tuesday</td>
        <td>CSS</td>
    </tr>
    <tr>
        <td>Wednesday</td>
        <td>JavaScript</td>
    </tr>
</table>

Table with Column Span (Merging Columns)

<table border="1">
    <tr>
        <th colspan="2">Student Info</th>
    </tr>
    <tr>
        <td>Name</td>
        <td>Helen</td>
    </tr>
    <tr>
        <td>Grade</td>
        <td>10</td>
    </tr>
</table>
Result with colspan:
Student Info
Name Helen
Grade 10

Table with Row Span (Merging Rows)

<table border="1">
    <tr>
        <th rowspan="2">Name</th>
        <td>Abebe</td>
    </tr>
    <tr>
        <td>Bekele</td>
    </tr>
</table>

7. Practice Exercises

📝 Exercise 1: Registration Form

Create an HTML form that collects:

  • Full Name (text field)
  • Email (email field)
  • Age (number field)
  • Gender (radio buttons)
  • City (dropdown)
  • Submit button

📊 Exercise 2: Class Marksheet Table

Create a table with border="1" showing:

  • Columns: Student Name, Maths, English, Science
  • At least 4 students
  • Add a header row with <th>

📅 Exercise 3: Weekly Timetable

Create a table showing your school schedule:

  • Rows: Monday to Friday
  • Columns: Time, Subject, Teacher

⭐ Challenge: Feedback Form with Table

Combine both! Create a feedback form that collects name, email, rating (dropdown), and message (textarea). Below the form, display a sample "Previous Feedback" table showing 3 example feedback entries.

💡 Tip for Students

Save your HTML file and open it in Chrome to test. You don't need a server! Forms won't submit data yet (that needs PHP later), but you can see how they look and work.

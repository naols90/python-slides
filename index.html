📌 Exercise 1 · simple if
🧪 Task: Write a Python program that checks if a number is positive.
Given temperature = 5, if temperature is greater than 0, print "❄️ Above freezing point".
(only if block, no else) 
# starter code
temperature = 5
if temperature > 0:
    print("Above freezing point")
✨ Click to see output
💡 Solution · if statement
# Python solution
temperature = 5
if temperature > 0:
    print("✅ Above freezing point")
# Output: ✅ Above freezing point
📖 Explanation: Condition 5 > 0 is True, so the indented block runs.
In Python, indentation is mandatory.
🔄 Exercise 2 · if-else
🎫 Task: Check voting eligibility.
Create variable age = 16. If age >= 18, print "🗳️ Eligible to vote", otherwise print "📵 Not eligible". 
age = 16
if age >= 18:
    print("Eligible to vote")
else:
    print("Not eligible")
⚡ Press to test
🧩 Solution · if-else logic
age = 16
if age >= 18:
    print("Eligible to vote")
else:
    print("Not eligible")   # ✅ Output: Not eligible
💡 Why? 16 is not >= 18 → else block executes.

📊 Exercise 3 · if-elif-else
🎓 Task: Grade classifier.
Given score = 85, assign grade letter:
A (>=90), B (>=75), C (>=60), F (otherwise).
Print the final grade. 
score = 85
if score >= 90:
    grade = "A"
elif score >= 75:
    grade = "B"
elif score >= 60:
    grade = "C"
else:
    grade = "F"
print(f"Grade: {grade}")
📖 Click to compute
🏆 Solution · elif chain
score = 85
if score >= 90:
    grade = "A"
elif score >= 75:   # ✅ True (85 >= 75)
    grade = "B"
elif score >= 60:
    grade = "C"
else:
    grade = "F"
print(f"Grade: {grade}")   # Output: Grade: B
🔎 Explanation: elif checks in order; first true block executes.
Score 85 meets score >= 75 → grade "B".
🪹 Exercise 4 · nested conditions
🎟️ Task: Museum discount logic.

If not member → "No discount".

✨ Click to see result
🧬 nested if solution
is_member = True
amount = 120
if is_member:
    if amount > 100:
        discount = "20% super discount"   # ✅ executes
    else:
        discount = "10% discount"
else:
    discount = "No discount"
print(f"Discount: {discount}")   # Output: 20% super discount
📘 Logic: outer if checks membership → inner if checks amount threshold.
Nested conditions are perfect for multi-level decisions.

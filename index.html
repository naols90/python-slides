Simple Calculator in Python variables + if / elif / else

Take two numbers from the user, choose operation (+, -, *, /) · Handle division by zero · No functions, no return

Python Code (Beginner Friendly) no def, no return 
# SIMPLE CALCULATOR
# Using only variables and if-elif-else statements

# Step 1: Get two numbers from the user
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))

# Step 2: Display menu and get operation choice
print("\\nSelect operation:")
print("1. Addition (+)")
print("2. Subtraction (-)")
print("3. Multiplication (*)")
print("4. Division (/)")

choice = input("Enter choice (1/2/3/4): ")

# Step 3: Perform calculation using if-elif-else
if choice == '1':
    result = num1 + num2
    operation = "+"
elif choice == '2':
    result = num1 - num2
    operation = "-"
elif choice == '3':
    result = num1 * num2
    operation = "*"
elif choice == '4':
    # Check division by zero
    if num2 == 0:
        print("ERROR: Cannot divide by zero!")
        result = None
    else:
        result = num1 / num2
        operation = "/"
else:
    print("Invalid choice! Please select 1, 2, 3, or 4.")
    result = None

# Step 4: Show the result (if valid)
if result is not None:
    print(f"\\n{num1} {operation} {num2} = {result}")

📚 What students learn here:

✅ Variables – num1, num2, choice, result, operation
✅ if / elif / else – to choose the right math operation
✅ Nested if – inside division to check zero
✅ input() and float() – get numbers from user
✅ Division by zero protection – prevents crash

No functions, no return statements – just sequential code that any beginner can follow.
